MAY 2023 - PRESENT
This project is an ongoing collaboration with Groove Into The Void, a community-based cultural exchange based in Karachi, Pakistan.
For "Controlled Chaos", which featured German-based artist Radical Softness, I live-coded audio-reactive visuals using JavaScript and Hydra.js. The improvised nature of this 8-hour long performance allowed me to engage with the audience and the music performers in real-time, which led to a more immersive experience.
For "Hyperspace", which featured UK-based artist Sam Binga, I created several animated posters that featured kinetic typography, motion graphics and real-time visuals generated using JavaScript.
Through this collaboration, I have been able to gain a deeper understanding of how people engage and interact with the artwork, and how I can apply my passion for creative technology towards building connections that are rooted in community.
